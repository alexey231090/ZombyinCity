Формат файла ArchitectureVisualizer.ai.json для AI:

{
  "items": [
    {
      "pathName": "Название цепочки",
      "description": "Описание цепочки",
      "steps": [
        {
          "scriptName": "ПолноеИмяСкрипта",
          "variableNames": ["имяПеременной1", "имяПеременной2"],
          "comment": "Комментарий к шагу"
        }
      ]
    }
  ]
}

Правила:
- Все новые пути добавлять в массив items.
- Не использовать комментарии // или /* ... */.
- Не добавлять лишние поля в корневой объект.
- Пример выше можно копировать и изменять.
- Файл должен быть в кодировке UTF-8.

Пример рабочего пути:

{
  "items": [
    {
      "pathName": "Отслеживание жизни зомби",
      "description": "Путь для дебага переменной жизни зомби.",
      "steps": [
        {
          "scriptName": "FpsZomby.Zombi",
          "variableNames": ["lifeZombi"],
          "comment": "Текущее значение жизни зомби."
        }
      ]
    }
  ]
}
